## 三者关系


这三篇是 Anthropic 同一批 Applied AI / Engineering 团队写的，互相交叉引用，构成一个完整的 Agent 工程闭环：


- **工具设计**（03）是地基——定义 Agent 与世界交互的契约。

- **上下文工程**（02）是中层——管理"什么信息进入有限注意力预算"，其中 sub-agent 一节直接引向多 Agent。

- **多 Agent 编排**（01）是上层——当任务超出单 Agent 单上下文时的扩展架构，其工具设计原则又回指 03。


阅读/检索顺序建议：03 → 02 → 01（地基到上层），或按当前要解决的问题直接定位。
